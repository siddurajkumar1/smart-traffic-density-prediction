# Smart Traffic Density Prediction System

## Overview
This project predicts traffic density using Machine Learning (Logistic Regression).

## Features
- Traffic Density Prediction
- Interactive Dashboard
- ROC Analysis
- Classification Metrics
- Live Prediction
- Responsive UI

## Technologies Used
- React
- TypeScript
- Vite
- Python
- Scikit-learn
- Tailwind CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/siddurajkumar1/smart-traffic-density-prediction.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
