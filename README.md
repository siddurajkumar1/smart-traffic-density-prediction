# Smart Traffic Density Prediction System

## Overview
This project predicts traffic density levels using Machine Learning (Logistic Regression). It provides real-time traffic density prediction through an interactive dashboard and visualization tools.

## Features
- 🚦 Traffic Density Prediction
- 📊 Interactive Dashboard
- 📈 ROC Curve Analysis
- 📉 Classification Metrics
- 🔴 Live Prediction System
- 📱 Responsive User Interface

## Technologies Used

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Python
- Scikit-learn
- Logistic Regression
- Power BI

## Installation

### Clone the Repository

```bash
git clone https://github.com/siddurajkumar1/smart-traffic-density-prediction.git
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```
